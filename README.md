# GoBarber

## Sobre o Projeto
Esse projeto tem como finalidade a criação de um sistema composto de uma API, App Web e App Mobile para a nomeação e agendamento de barbeiro.

## Tecnológias Usadas

  ### Padronização de Código
  * **Eslint**
  * **EditorConfig**
  * **Prettier**

  ### API
  * **NodeJS**
  * **Express**
  * **Sequelize**
  * **PostgreSQL**
  * **Mongoose**
  * **MongoDB**
  * **Bee-queue**
  * **Nodemailer**
  * **Sentry**
